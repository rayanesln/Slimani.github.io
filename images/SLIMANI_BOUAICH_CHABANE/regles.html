Configuration du Match

Avant le match, l'utilisateur choisit une formation tactique qui définit les positions de départ :

  • 4-4-2 : Formation classique et équilibrée.
  • 4-3-3 : Formation à vocation offensive.
  • 5-3-2 : Formation défensive et contre-attaque.

Règles Simplifiées

Pour assurer la fluidité de la simulation, les règles officielles ont été adaptées :

La Touche
Si le ballon sort latéralement, il est automatiquement replacé sur la ligne pour une remise en jeu.
Le Corner
Si le ballon sort derrière la ligne de but touché par un défenseur, un tir est généré depuis le coin.
Les Duels
Lors d'une collision entre deux joueurs, le système compare les statistiques. Si le défenseur gagne, il récupère la possession. Sinon, l'attaquant continue sa course.
