## Sujet du Site
Ce site Web présente les spécifications fonctionnelles et techniques de notre projet de simulation de football autonome intégrant la gestion de super-pouvoirs (Magic Match). Il détaille les règles sportives, l'architecture technique de l'intelligence artificielle et présente l'équipe de développement.
